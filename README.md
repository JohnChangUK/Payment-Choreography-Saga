## Saga Pattern - Payment Microservices

### Implemented with 
- Spring Cloud Stream with Apache Kafka Binder
- Reactive Spring

#### To start ZooKeeper and Kafka Brokers
```
cd docker
```

- #### Run 
```
docker-compose up
```
